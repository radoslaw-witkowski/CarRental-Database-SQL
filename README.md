# Car Rental Database (Oracle SQL)

A complete relational database project designed for a car rental company.  
The project includes tables, relationships, constraints, sample data and documentation.

## Contents
- Database schema (tables, PK/FK, integrity constraints)
- Many-to-many and one-to-many relationships
- INSERT scripts with sample realistic data
- COMMENT ON descriptions for documentation
- Full business domain:
  - customers
  - vehicles
  - rentals
  - payments
  - insurance policies

## Technologies
- Oracle SQL
- Relational database design

## How to use
1. Run the SQL script in Oracle SQL Developer or any compatible environment.
2. All tables, data and constraints will be created automatically.

## Notes
This project was created as a real-world style business system to practice SQL database design.
